# Changelog

All notable changes to forge-sutra are documented here.

## [1.0.0] - 2026-05-30

### Added
- **Phase 0:** Core scan + view — structural graph for TS/JS repos
- **Phase 1:** External-host allowlist, dynamic-segment resolver, `feature.sutra.md` contracts, graph diff, scaffold, watch, cross-repo reconcile
- **Phase 2:** CLI polish (`forge-sutra` + `sutra` bins), graph migration (v0→v1), benchmark + `--profile`, multi-file contracts, contract drift + reconcile view panels, npm publish prep
- **Phase 3:** Contract summary in view, export command (contracts/schema/issues), feature filter
- **Phase 4:** GitHub Actions CI, regression guard script
- **Phase 5:** JSON Schema export for graph.json, issues CSV/JSON export
- **Phase 6:** `.sutra/hooks.json` post-scan hook runner

### Claim bounds (unchanged)
All results are **heuristic / candidate** — static approximation only. Not auto-debug, not runtime verification.
